# Трамплин — карьерная платформа

Интерактивная карьерная платформа для студентов, выпускников 
и работодателей в сфере IT. Централизованное взаимодействие: 
вакансии, стажировки, менторские программы, карьерные мероприятия.

## Стек технологий

- **Frontend**: React, Next.js 14 (App Router, SSR), Tailwind CSS, Leaflet
- **Backend**: Python 3.12, FastAPI, SQLAlchemy 2.0 (async)
- **База данных**: PostgreSQL 16 + PostGIS, Redis 7
- **Инфраструктура**: Docker, Docker Compose, MinIO

## Роли пользователей

- **Гость** — просмотр карты и списка вакансий, фильтры, избранное
- **Соискатель** — профиль, резюме, отклики, контакты, приватность
- **Работодатель** — публикация вакансий (после верификации), управление откликами
- **Куратор** — верификация компаний, модерация карточек, управление тегами
- **Администратор** — все права куратора + создание других кураторов

## Быстрый старт

### Требования
- Docker Desktop
- Docker Compose

### Запуск

1. Клонируй репозиторий:
   git clone https://github.com/AlexanderBannikovIT/trampliin-platform.git
   cd trampliin-platform

2. Создай .env файл в корне (скопируй из .env.example):
   cp .env.example .env

3. Запусти проект:
   docker compose -f docker-compose.dev.yml up -d

4. Создай администратора и тестовые данные:
   docker compose -f docker-compose.dev.yml exec backend python -m app.core.seed

5. Открой в браузере:
   - Платформа: http://localhost:3000
   - API документация: http://localhost:8000/docs
   - Adminer (БД): http://localhost:8080

## Функциональность

- Интерактивная карта с маркерами вакансий (Leaflet + OpenStreetMap)
- Два режима отображения: карта и список
- Фильтрация по типу, формату, зарплате, городу, тегам навыков
- Попап при наведении и превью при клике на маркер
- Система откликов с отслеживанием статусов
- Профессиональные контакты между соискателями
- Настройки приватности профиля
- Верификация работодателей куратором
- Модерация карточек возможностей
- 43 предустановленных тега навыков
- Адаптивный дизайн (мобиль, планшет, десктоп)
- JWT аутентификация в httpOnly cookie
- Redis кэширование поисковых запросов
- PostGIS для геопространственных запросов
